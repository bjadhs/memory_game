## Features

## How to Run

1.  **Install**:
    ```bash
    npm install
    ```

2.  **Play**:
    ```bash
    npm run dev
    ```

## How to Play
- Click a card to reveal the fruit.
- Click another card to find its match.
- If they match, they stay flipped! If not, they flip back.
